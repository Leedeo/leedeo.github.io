Normas de la Comunidad

Última actualización: 29 de Marzo, 2026

{% comment %}Marco general antes de enumerar reglas concretas.{% endcomment %}

Bienvenida

En Leedeo Studio busco fomentar un espacio de aprendizaje, colaboración y respeto. Tanto en los comentarios del blog como en mis redes sociales, estas normas existen para asegurar que todos los creadores se sientan seguros y motivados.

{% comment %}Norma base de convivencia para todos los canales del proyecto.{% endcomment %}

1. Respeto y Educación

Esta es la base de todo. No se tolerarán:

  • Insultos, acoso o ataques personales a otros usuarios o a mí.
  • Discursos de odio, racismo, sexismo o cualquier tipo de discriminación.
  • Actitudes tóxicas que busquen desanimar a los demás.
{% comment %}Criterios para mantener comentarios y espacios útiles.{% endcomment %}

2. No al Spam ni contenido Irrelevante

Para mantener el canal limpio y útil:

  • No publiques enlaces de publicidad no solicitada o "spam" de tus proyectos de forma masiva.
  • Evita repetir el mismo comentario varias veces.
  • No utilices los comentarios del blog para pedir soporte técnico ajeno al post. Usa los canales adecuados para ello.
{% comment %}Separación entre crítica técnica válida y comportamiento destructivo.{% endcomment %}

3. Crítica Constructiva

Fomento el debate técnico. Si no estás de acuerdo con algo, exprésalo con argumentos. Las críticas destructivas que no aporten valor serán eliminadas para mantener la calidad del ecosistema de aprendizaje.

{% comment %}Protección mínima de datos personales de terceros.{% endcomment %}

4. Privacidad (No Doxxing)

Respeta la privacidad ajena. No compartas información personal (correos, teléfonos, direcciones) de otros usuarios sin su consentimiento explícito.

{% comment %}Límites de contenido para conservar el carácter educativo del espacio.{% endcomment %}

5. Contenido Apropiado

Este es un espacio profesional y educativo. No se permite contenido explícito (NSFW), violento o ilegal.

{% comment %}Cierre con el criterio de moderación aplicado por el responsable del sitio.{% endcomment %}

Moderación

El incumplimiento de estas normas puede resultar en la eliminación del comentario o el bloqueo permanente de la cuenta. Como responsable de Leedeo Studio, tengo la palabra final sobre lo que se considera apropiado.
